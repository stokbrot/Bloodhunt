What is this?

An color Aimbot for Apex Legends :

featuring 2 Modes: 1-Rage and 2-Legit

It works with a thermal-thread scope and with the Bloodhound Ult

It is safe to use and undetectable:

- Random application name

- No read/writing of the games memory

- uses mouse_event so it is not recogniseable by the anticheat

How It Works

It scannes for a range of the color the thermal/Bloodhound ult uses

then it detects the higest color, calculates the middle and then minus some pixels

this is roughly the center of the head

finaly it aims there
